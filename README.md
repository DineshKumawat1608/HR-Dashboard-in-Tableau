HR Dashboard in Tableau:- Making Employee Data Easy to Understand

 I recently built this HR Analytics Dashboard in Tableau to make employee data simple and visual.
 It helps HR teams quickly see what’s going on — who’s hired, who left, and how the team looks overall.

What’s inside the dashboard:-
 - Overview of Active, Hired, and Terminated employees
 - Department & Location wise breakdown
 - Gender and Education distribution
 - Salary comparison by Education, Age, and Gender
 - An Employee Details Table showing role, salary, location, and experience

Why I made this:-
 I wanted to show how HR data can tell powerful stories — like hiring trends, pay differences, or which departments need attention.
 Instead of long reports, this dashboard gives a clear picture in seconds.

Tools & Skills Used:-
 - Tool: Tableau
 - Data: Created using ChatGPT + Faker library
 - Skills: Filters, parameters, calculated fields, actions, and layout design

This project helped me understand how to turn raw HR data into something visual and useful for real decisions.
